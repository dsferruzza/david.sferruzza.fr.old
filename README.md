david.sferruzza.fr
==================

This is my personnal website's source code.
It is made with [Hakyll](http://jaspervdj.be/hakyll/).

## How to install

- install [stack](http://haskellstack.org)
- clone this repo
- `stack build`

To build: `stack exec david-sferruzza-fr build`
